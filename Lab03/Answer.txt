1. When should accessor methods be used?
	Chúng ta cần sử dụng accessor methods khi:
	+ Khi muốn bảo vệ dữ liệu khỏi việc truy vấn trực tiếp từ bên ngoài, duy trì tính đóng gói
	+ Khi cần kiểm soát truy nhập: Có thể them vào các logic kiểm tra, xác thực
2. If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
	Có, java cho phép có nhiều phương thức khởi tạo.
